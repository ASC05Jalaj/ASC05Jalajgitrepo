Welcome, Administrator

Use the navigation menu to manage the system's key components: trips, cabs, bookings, payments, and feedback.

Manage Trips

View, add, and update trips for the users.

  • Active Trips: Monitor all ongoing trips.
  • Cancel Trips: Allow users to cancel their trips.
  • Update Trip Details: Modify trip schedules or destinations.
  • Generate Reports: View trip reports based on user feedback.

Manage Cabs

Add and manage cabs for the trips.

  • Available Cabs: View all cabs that are available for trips.
  • Add New Cab: Add new cabs to the fleet.
  • Update Cab Information: Modify cab details like model, capacity, and driver information.
  • Track Cab Usage: Monitor the usage and location of cabs.

Manage Bookings

Oversee and manage all bookings made by users.

  • View Active Bookings: Check the list of active bookings.
  • Cancel Bookings: Allow users to cancel their bookings if required.
  • Update Booking Status: Modify the status of bookings (confirmed, pending, etc.).
  • Booking History: View historical data of all past bookings.

Manage Payments

Track and manage payments for trips and bookings.

  • View Payment History: Check past payments made by users.
  • Manage Pending Payments: Monitor payments that are yet to be completed.
  • Transaction Reports: Generate reports for payment-related transactions.
  • Payment Status: Update payment status and send notifications to users.
